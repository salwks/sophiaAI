# 답변 품질 평가 기준

## 1. 필수 요소 (Must Have)

### ✅ 정확성 (Accuracy)
- [ ] 의학적으로 정확한 정보
- [ ] 구체적인 수치/기준 제시
- [ ] 최신 가이드라인 반영

### ✅ 출처 (Sources)
- [ ] 참고 문헌 명시 ([1], [2] 형식)
- [ ] BI-RADS 가이드라인 출처 표시
- [ ] 출처 클릭 가능 (PubMed 링크)

- [ ] "의학적 조언 아님" 명시
- [ ] "연구 정보 제공" 명확화

## 3. 카테고리별 평가 기준

### BI-RADS 카테고리 질문

**예시: "BI-RADS 5는 무엇인가요?"**

✅ **우수한 답변 필수 요소:**
1. 정확한 악성 확률 (≥95%)
2. 조직검사 권고사항
3. 영상 특징 설명
4. Category 4와의 차이
5. 출처 명시

✅ **실제 우수 답변 예시:**
```
BI-RADS Category 5는 악성 종양의 가능성이 ≥95%로 평가되는 소견입니다.[1]

주요 특징:
- 불규칙한 경계의 spiculated mass
- 미세석회화를 동반한 고밀도 종괴
- 새로운 선형/분지형 석회화

관리:
- 조직검사 필수 권장
- 비악성 조직검사 결과는 불일치로 간주
- 일반적으로 수술적 생검 권장[1]

이 정보는 ACR BI-RADS 가이드라인 기반이며, 개인의 의학적 조언은 아닙니다.
```

❌ **불량한 답변 예시:**
```
BI-RADS 5는 암이 의심되는 경우입니다. 검사가 필요합니다.
```

### 기술 비교 질문

**예시: "DBT와 mammography 차이는?"**

✅ **우수한 답변 필수 요소:**
1. 기술적 차이 (3D vs 2D)
2. 임상적 장점
3. 제한점
4. 적용 대상
5. 연구 근거

✅ **답변 구조:**
```
1. 기술 차이
   - DBT: 3D 단층영상
   - FFDM: 2D 평면영상

2. 장점
   - DBT: 조직 겹침 감소, 병변 검출 향상
   - 근거: [연구 인용]

3. 제한점
   - DBT: 방사선 노출 증가, 검사 시간

4. 적용
   - Dense breast에서 유용
   - Screening & diagnostic 모두 가능

출처: [1], [2]
```

### 임상 가이드라인 질문

**예시: "맘모그래피 스크리닝은 몇 살부터?"**

✅ **우수한 답변 필수 요소:**
1. 주요 가이드라인별 권고 (ACR, USPSTF)
2. 연령별 권고 차이
3. 고위험군 예외
4. 근거 제시
5. 논쟁점 설명

✅ **답변 예시:**
```
맘모그래피 스크리닝 시작 연령은 가이드라인마다 다릅니다:

1. ACR (American College of Radiology)
   - 40세부터 매년 권장[1]
   - 근거: 40대 사망률 감소 효과

2. USPSTF
   - 50세부터 2년마다 권장[2]
   - 40-49세: 개인별 결정

3. 고위험군
   - 가족력: 더 이른 시작
   - BRCA 변이: 25-30세부터 MRI 추가

개인의 위험도에 따라 전문의와 상담이 필요합니다.
```

## 5. 개선이 필요한 답변 유형

### 🔴 Type 1: 모호한 답변
**문제:**
> "BI-RADS 5는 암이 의심되는 경우입니다."

**개선:**
> "BI-RADS Category 5는 악성 가능성이 ≥95%로 평가되는 소견으로, 조직검사가 필수적으로 권장됩니다.[1]"

---

### 🔴 Type 2: 출처 없는 답변
**문제:**
> "DBT는 더 정확합니다."

**개선:**
> "DBT는 전통적 mammography 대비 병변 검출률이 20-65% 향상되었습니다.[1,2]"

---

### 🔴 Type 3: 주의사항 누락
**문제:**
> "40세부터 매년 검사하세요."

**개선:**
> "ACR 가이드라인은 40세부터 매년 검사를 권장합니다.[1] 개인의 위험도에 따라 전문의와 상담하시기 바랍니다. 이는 연구 정보이며 개인의 의학적 조언은 아닙니다."

## 7. 실제 평가 예시

### 질문: "BI-RADS 5는 무엇인가요?"

### 답변 A (실제 시스템 답변)
> "BI-RADS 5는 유방영상 검사에서 악성 종양의 가능성이 매우 높다고 판단되는 경우에 주어지는 분류입니다. 이 분류는 악성종양의 가능성은 최소 95% 이상으로 추정될 때 사용되며, 일반적으로 이러한 특징을 가진 유방영상 소견은 조직검사를 통해 확실한 진단이 필요하다고 판단됩니다.[1]"

**평가:**
- ✅ 정확성: 40/40 (95% 명시, 조직검사 언급)
- ✅ 완전성: 20/25 (핵심 정보 있으나 영상 특징 상세 부족)
- ✅ 출처: 15/15 ([1] 명시)
- ✅ 구조: 8/10 (읽기 쉬우나 구조화 부족)
- ✅ 안전성: 10/10 (주의사항 명시)

**총점: 93/100** 🟢 우수

**개선 제안:**
- 영상 특징 구체화 (spiculated mass, 미세석회화 등)
- 구조화된 형식 (번호, 불렛 포인트)

## 9. 테스트 시나리오

### 시나리오 1: BI-RADS 정의
**입력:** "BI-RADS 4A는 무엇인가요?"
**기대 출력:**
- ✅ 2-10% 악성 가능성
- ✅ Low suspicion
- ✅ 조직검사 권장
- ✅ 4B, 4C와의 차이

### 시나리오 2: 기술 비교
**입력:** "DBT vs FFDM"
**기대 출력:**
- ✅ 3D vs 2D 차이
- ✅ 장단점 비교표
- ✅ 임상 적용
- ✅ 연구 근거

### 시나리오 3: 임상 가이드라인
**입력:** "맘모그래피 검사 주기는?"
**기대 출력:**
- ✅ 가이드라인별 차이 (ACR vs USPSTF)
- ✅ 연령별 권고
- ✅ 고위험군 예외
- ✅ 개인별 상담 권유
