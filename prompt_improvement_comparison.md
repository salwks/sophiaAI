# 프롬프트 개선 전후 비교

## 개선 사항 요약

### 기존 프롬프트 (Before)
```
당신은 유방영상의학 전문 AI 어시스턴트입니다.

중요한 규칙:
1. 제공된 논문과 BI-RADS 가이드라인 내용만을 기반으로 답변하세요
2. 추측하지 말고, 제공된 정보에 없으면 명시하세요
3. 답변 시 반드시 출처를 명시하세요
4. 의학적 조언이 아닌 연구 정보 제공임을 명확히 하세요
5. 한국어로 답변하세요
```

**문제점:**
- ❌ 구체성 부족 ("정확한 수치" 요구 없음)
- ❌ 구조화 지침 없음
- ❌ 질문 유형별 가이드 없음
- ❌ 답변 형식 예시 없음

---

### 개선된 프롬프트 (After)
```
당신은 유방영상의학 전문 AI 어시스턴트입니다.

핵심 원칙:
1. 제공된 논문과 BI-RADS 가이드라인 내용만을 기반으로 답변
2. 정확한 의학 용어와 구체적인 수치 필수 사용  ← NEW
3. 추측 금지 - 정보 없으면 명확히 명시
4. 모든 주장에 출처 표시 ([1], [2] 형식)
5. 한국어로 답변

답변 품질 기준:  ← NEW
- 정확성: 구체적인 수치, 퍼센트, 기준 명시 (예: "≥95%", "2-10%")
- 완전성: 정의, 특징, 관리 권고사항 모두 포함
- 구조화: 불렛 포인트(•), 단락 구분으로 가독성 확보
- 간결성: 불필요한 반복 제거, 핵심 정보 위주

질문 유형별 필수 포함 사항:  ← NEW
- BI-RADS 카테고리: 정확한 악성 확률(%), 영상 특징, 관리 권고
- 기술 비교: 핵심 차이점, 장단점, 임상 적용
- 가이드라인: 주요 기관별 권고(ACR, USPSTF 등), 근거

+ 답변 형식 예시 제공  ← NEW
```

**개선점:**
- ✅ 구체적 수치 요구 ("≥95%", "2-10%")
- ✅ 구조화 가이드 (불렛 포인트, 단락)
- ✅ 질문 유형별 체크리스트
- ✅ 답변 템플릿 예시 제공

## 실제 테스트 비교

### 질문: "BI-RADS 5는 무엇인가요?"

#### Before (개선 전) - 80점
```
BI-RADS 5는 유방영상 검사에서 악성 종양의 가능성이 매우 높다고
판단되는 경우에 주어지는 평가입니다. 이 평가는 암의 확률이
대략적으로 95% 이상일 때 사용되며, 이러한 평가를 받은 발견들은
대부분 비정형적이고 특징적인 악성 종양의 징후를 보입니다...
```

**문제:**
- ⚠️ 안전성 문구 누락
- ⚠️ 구조화 부족 (한 덩어리)
- ⚠️ 장황함

#### After (개선 예상) - 95점
```
BI-RADS Category 5는 악성 가능성이 ≥95%로 평가되는 소견입니다.[1]

**주요 특징:**
• 불규칙한 경계의 spiculated mass
• 미세석회화를 동반한 고밀도 종괴
• 새로운 선형/분지형 석회화[1]

**관리:**
조직검사 필수 권장. 비악성 결과는 불일치로 간주되어
재검사(보통 수술적 생검) 필요[1]

이 답변은 연구 정보를 기반으로 제공되며,
개인의 건강에 대한 의학적 조언은 아닙니다.
```

**개선:**
- ✅ 정확한 수치 (≥95%)
- ✅ 구조화된 형식
- ✅ 안전성 문구 포함
- ✅ 간결하고 명확

### 즉시 테스트할 질문
1. BI-RADS 5는 무엇인가요?
2. BI-RADS Category 4A는 무엇인가요?
3. breast density는 무엇인가요?
4. DBT와 mammography 차이는?
5. 맘모그래피 스크리닝은 몇 살부터 하나요?

### 평가 기준
각 답변에서 확인:
- [ ] 구체적인 수치 포함 (%, 연령 등)
- [ ] 불렛 포인트로 구조화
- [ ] 정의 + 특징 + 관리 모두 포함
- [ ] 출처 표시
- [ ] 안전성 문구

## 장기적 개선 방안

### 1. 프롬프트 지속 개선
- 사용자 피드백 수집
- 전문의 검토 결과 반영
- 월 1회 프롬프트 리뷰

### 2. 답변 품질 모니터링
- 자동 평가 스크립트 정기 실행
- 품질 점수 추이 추적
- 낮은 점수 답변 분석

### 3. 도메인별 템플릿 확장
- BI-RADS 카테고리: 현재 템플릿 사용
- 기술 비교: 표 형식 템플릿 추가
- 가이드라인: 기관별 비교 템플릿 추가
- 임상 케이스: 단계별 접근 템플릿 추가

## 결론

**핵심 개선:**
1. 구체적 수치 요구 → 정확성 향상
2. 구조화 가이드 → 가독성 향상
3. 질문별 체크리스트 → 완전성 향상
4. 답변 템플릿 → 일관성 향상

**예상 결과:**
- 평균 점수: 74.2 → **93점** (+18.8)
- 등급: 🟡 양호 → **🟢 우수**
